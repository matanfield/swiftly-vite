# Configuration

Learn how to configure your VitePress site.

## Basic Config

The essential configurations you'll need:

```ts
// .vitepress/config.ts
export default {
  // site-level config
  title: 'My Site',
  description: 'A VitePress Site',
  
  // theme-level config
  themeConfig: {
    // ...
  }
}
```
